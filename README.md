# BD_exemplos
bases de dados para teste

## Configuração

**Importante:** (no confic.toml)
- No campo da password, tem de colocar a password do seu MySQL Workbench
- Sempre que quiser dar import a uma base de dados nova, tem de mudar o nome da BD no código

**Para Importar
python (ou python3 depende de como tiverem instalado) nome_ficheiro.py 
por exemplo python3 teste2_seed.py
